# Sales Performance Dashboard

## Overview

This project is an interactive Sales Performance Dashboard developed using Microsoft Power BI. It provides insights into sales performance, profitability, order volume, and category-wise distribution using the Superstore dataset.

## Dashboard Preview

![Sales Dashboard](dashboard.png)

## Features

- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales Trend Analysis (YTD vs MTD)
- Sales & Profit by Region
- Sales by Category
- Category-wise Sales Distribution
- Interactive Filters (Category, Region, Order Date)

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel
